# 📂 Datasets

This project integrates multiple global datasets to simulate adaptive regional dynamics under resource scarcity, climate variation, and international trade interactions.
## 🌧️ Global Precipitation Modeling Dataset

Source: OEMC Hackathon – Global Precipitation Modeling
🔗https://data.unodc.org/

Purpose in Project

Climate variability modeling

Rainfall-driven resource fluctuations

Drought and extreme weather event simulation

Environmental shock generation for agents

This dataset provides global precipitation information used to dynamically modify regional resource availability during simulation cycles.

## 🌍 Global Commodity Trade Statistics (UN Comtrade)

Source: United Nations Global Commodity Trade Statistics
## worldbank data

Purpose in Project

Modeling inter-region trade relationships

Import/export dependency analysis

Resource exchange simulation

Trade network construction between agents

The dataset enables creation of dynamic economic connections between simulated regions.

## 🔄 International Trade Database

Source: International Trade Database (UN Comtrade Derived)
🔗 https://www.kaggle.com/datasets/appetukhov/international-trade-database

Purpose in Project

Bilateral trade network generation

Region-to-region trade flow analysis

Agent negotiation and cooperation modeling

This dataset provides structured country-pair trade relationships suitable for network-based simulations.

## 🌾 FAOSTAT Crops and Livestock Dataset

Source: FAOSTAT Crops and Livestock Data
🔗 https://www.kaggle.com/datasets/vijayveersingh/faostat-crops-and-livestock-data

Purpose in Project

Food production estimation

Agricultural capacity modeling

Land utilization approximation

Regional sustainability indicators

Agricultural production metrics act as proxies for food resource availability.

## ⚡ World Development Indicators (World Bank)

Source: World Bank World Development Indicators
🔗 https://www.kaggle.com/datasets/nicolasgonzalezmunoz/world-bank-world-development-indicators

Purpose in Project

Regional baseline initialization

Energy consumption and production proxies

Water availability indicators

Population and economic state modeling

This dataset forms the foundational socio-economic layer of the simulation environment.
